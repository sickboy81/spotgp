# Ícones PWA

Esta pasta deve conter os seguintes arquivos de ícone:

## Arquivos Necessários

- `icon-16x16.png` - Favicon pequeno
- `icon-32x32.png` - Favicon médio
- `icon-72x72.png` - Android
- `icon-96x96.png` - Android
- `icon-128x128.png` - Android
- `icon-144x144.png` - Windows
- `icon-152x152.png` - iOS
- `icon-180x180.png` - iOS (apple-touch-icon)
- `icon-192x192.png` - Android (mínimo PWA)
- `icon-384x384.png` - Android
- `icon-512x512.png` - Android (splash screen)
- `favicon.ico` - Favicon tradicional

## Como Gerar

Consulte `scripts/generate-icons.md` para instruções detalhadas.

## Nota Temporária

Enquanto os ícones não são gerados, o app funcionará, mas pode não aparecer corretamente quando instalado. Gere os ícones usando uma das opções descritas em `scripts/generate-icons.md`.
